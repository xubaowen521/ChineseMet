# Examples

This directory contains example scripts demonstrating how to use the ChineseMet benchmark framework.

## Scripts

| Script | Description |
|--------|-------------|
| `basic_evaluation.py` | Basic model evaluation pipeline |
| `analyze_by_question_type.py` | Separate metrics for single-choice vs multiple-choice |
| `generate_heatmap.py` | Generate comparison heatmaps |

## Usage

```bash
cd examples
python basic_evaluation.py
```

## Customization

All examples use relative paths and can be easily adapted to your environment by modifying the configuration parameters at the top of each script.
